Self-hosted font files for nEIP
================================

Place the actual font files here for offline/Docker deployment.

Required files:
- NotoSansThai-Regular.woff2   (Noto Sans Thai Regular 400)
- NotoSansThai-Medium.woff2    (Noto Sans Thai Medium 500)
- NotoSansThai-SemiBold.woff2  (Noto Sans Thai SemiBold 600)
- NotoSansThai-Bold.woff2      (Noto Sans Thai Bold 700)
- Inter-Regular.woff2          (Inter Regular 400)
- Inter-Medium.woff2           (Inter Medium 500)
- Inter-SemiBold.woff2         (Inter SemiBold 600)
- Inter-Bold.woff2             (Inter Bold 700)
- JetBrainsMono-Regular.woff2  (JetBrains Mono Regular 400)
- JetBrainsMono-Medium.woff2   (JetBrains Mono Medium 500)

Sources:
- Noto Sans Thai: https://fonts.google.com/noto/specimen/Noto+Sans+Thai
- Inter: https://fonts.google.com/specimen/Inter (or https://rsms.me/inter/)
- JetBrains Mono: https://www.jetbrains.com/lp/mono/

Until actual font files are placed here, the system will fall back to
system fonts (ui-sans-serif, system-ui, etc.) which is fully functional.
